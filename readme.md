# FinControl Telegram Bot

Telegram бот для интеграции с финансовым приложением FinControl.
Обеспечивает привязку аккаунта и базовую регистрацию пользователей.

## Реализованный функционал

### 1. Привязка аккаунта
- Обработка `/start` с deep link: `t.me/bot?start={user_id}`
- Сохранение `telegram_id` в таблицу `users`
- Два способа регистрации:
  - По номеру телефона (создание нового пользователя)
  - По deep link (привязка к существующему)

### 2. База данных
Таблица `users`:
- `id` — первичный ключ
- `telegram_id` — ID пользователя в Telegram (уникальный)
- `username` — имя пользователя
- `phone` — номер телефона
- `created_at` — дата регистрации

### 3. Клавиатуры
- **Reply**: запрос номера телефона
- **Inline**: главное меню (4 кнопки: Баланс, Операции, Подписки, Профиль)

## API БД (db_queries.py)

| Функция                     | Параметры                                   | Возвращает    | Описание                                                |
|-----------------------------|---------------------------------------------|---------------|---------------------------------------------------------|
| get_user_by_telegram_id     | telegram_id: int                            | dict или None | Получить пользователя по Telegram ID                    |

## Сценарии использования

### Сценарий 1: Новый пользователь
1. Пользователь пишет /start
2. Бот предлагает отправить номер телефона
3. Пользователь отправляет номер
4. Создается запись в users с telegram_id
5. Бот показывает главное меню

### Сценарий 2: Привязка из приложения
1. Пользователь в приложении получает ссылку t.me/bot?start=123
2. Переходит по ссылке в Telegram
3. Бот находит пользователя с id=123
4. Сохраняет telegram_id в запись пользователя
5. Бот показывает главное меню

### Сценарий 3: Повторный вход
1. Пользователь пишет /start
2. Бот находит telegram_id в БД
3. Приветствует и показывает главное меню
