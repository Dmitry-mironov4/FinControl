# FinControl — flet-app
> Ветка с реализацией мобильного приложения на Python + Flet

## Запуск
```bash
git clone https://github.com/Dmitry-mironov4/FinControl.git
cd FinControl

uv sync

### Desktop
uv run python main.py

### iOS (Нужен Flet)
uv run flet run --ios main.py

### Android (Нужен Flet)
uv run flet run --android main.py
```

## Требования

| Инструмент | Версия |
|---|---|
| `users` | Пользователи (email, phone, telegram_id, password_hash) |
| `categories` | Категории доходов и расходов |
| `transactions` | Все операции (доходы и расходы) |
| `goals` | Финансовые цели |
| `subscriptions` | Подписки с датами списания |

Пароли хранятся в виде `pbkdf2_hmac(sha256, password, salt, 100_000_iterations)` — без открытого текста.

## Безопасность

- Хэширование паролей: PBKDF2-HMAC-SHA256 с уникальной солью на каждого пользователя
- Параметризованные SQL-запросы во всех функциях `db_queries.py` (защита от инъекций)
- Данные хранятся локально на устройстве
