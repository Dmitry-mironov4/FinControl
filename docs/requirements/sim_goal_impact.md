# Сценарий использования: SIM‑2 — Влияние покупки на срок цели

**Вопрос пользователя:**  
«Если я куплю это, на сколько месяцев отодвинется моя цель?»

---

## Исходные данные (из БД и ввод пользователя)

| Параметр | Источник |
|----------|----------|
| Цель: желаемая сумма, текущие накопления, ежемесячный платёж | Из цели пользователя |
| Сумма покупки | Ввод пользователя |

---

## Алгоритм

1. **Текущий срок**  
   `remaining = target - current_savings`  
   `current_months = ceil(remaining / monthly_savings)`

2. **Проверка возможности покупки**  
   Если `purchase_amount > current_savings` → недостаточно накоплений.

3. **Новый срок**  
   `new_savings = current_savings - purchase_amount`  
   `remaining_new = target - new_savings`  
   `new_months = ceil(remaining_new / monthly_savings)`

4. **Сдвиг**  
   `delay = new_months - current_months`

---

## Результат

Сообщение: «Цель отодвинется на X месяцев (было Y, станет Z)» или «Недостаточно накоплений».
